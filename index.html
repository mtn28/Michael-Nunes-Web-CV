👨‍💻 Como estagiário na Findmore Solutions, estive envolvido num projeto de desenvolvimento focado na criação de uma interface entre o HubSpot e a integração do SharePoint. Neste projeto, estive concentrado na construção de uma Extensão de Interface de Utilizador personalizada em React. O meu trabalho envolveu desenvolver uma interface eficiente que permite sincronizar dados entre essas plataformas, utilizando funções serverless para garantir a segurança dos dados e melhorar a experiência do utilizador na gestão de ficheiros e dados.

📚 Visão Geral:

Este projeto utiliza as Extensões de UI do HubSpot para criar cartões personalizados no CRM, integrando dados do SharePoint de forma eficiente. O frontend foi desenvolvido em React e as funções serverless do HubSpot foram usadas para a integração backend, promovendo a sincronização de dados entre SharePoint e HubSpot CRM.

🎯 Objetivos do Projeto:

  • 🔄 Sincronização de Dados: Facilitar o fluxo de dados em tempo real entre SharePoint e HubSpot CRM.
  • 🖼️ Cartões Personalizados: Criar cartões intuitivos para gestão eficiente de ficheiros no HubSpot.
  • 🔒 Integração Segura: Garantir a segurança no acesso a dados do SharePoint através de funções serverless do HubSpot.

📐 Objetivos Adicionais:

  • 🔍 Trabalho Exploratório: Dominar as ferramentas e APIs do HubSpot CRM, explorar a API Microsoft Graph para funcionalidades do SharePoint.
  • 🏗 Arquitetura da Aplicação: Construir uma aplicação escalável e robusta utilizando React, integrando operações backend com funções serverless do HubSpot.
  • 📁 Integração com SharePoint: Gerir ficheiros do SharePoint diretamente na interface do HubSpot, garantindo um acesso seguro e eficiente.
  • 🎴 Apresentação de Ficheiros: Exibir informações de ficheiros no HubSpot CRM através de cartões personalizados.
  • 🆕 Carregar Novos Ficheiros: Melhorar a experiência do utilizador ao simplificar o processo de upload de documentos.
  • 🔐 Segurança de Dados: Implementar protocolos rigorosos para proteger informações sensíveis.
  • 🧪 Testes e Garantia de Qualidade: Garantir elevados padrões de qualidade através de rigorosos testes e processos de QA.

🌍 No âmbito da disciplina "Projeto Integrado 3", criámos uma aplicação web completa, incorporando frontend, backend e base de dados, desenvolvida com Node.js, Express, React, Bootstrap e PostgreSQL. O nosso objetivo principal foi simplificar a negociação de excedente energético e fornecer informações detalhadas sobre o processo de compra e venda de energia.

🔑 Funcionalidades Principais:

  • 🏠 Página Inicial Informativa: Explicação do conceito de excedente energético, orientando os utilizadores nos passos para comprar ou vender energia, com botões de login e registo.
  • 📝 Registo e Perfis de Utilizador: Registo como comprador ou vendedor, cada um com funcionalidades próprias, e ainda um perfil de administração disponível apenas na base de dados.
  • 👤 Perfil do Comprador: Visualização dos dados pessoais, possibilidade de comprar excedente energético e gestão dos contratos associados às compras.
  • 💼 Perfil do Vendedor: Adição de vendas de excedente energético, gestão das infraestruturas (adicionar e editar), e visualização dos contratos ativos.
  • 📊 Perfil do Admin: Estatísticas da plataforma, número de utilizadores, custos do website, detalhes de todas as compras, perfis de utilizadores e controlo geral da plataforma.
  • ⚙️ Gestão de Dados: Configurações de conta ajustáveis para cada tipo de utilizador.

🔧 Tecnologias Utilizadas:

  • 📱 Frontend: React.js, Bootstrap
  • 🖥️ Backend: Node.js, Express.js, Sequelize
  • 💾 Base de Dados: PostgreSQL

🚀 Este projeto reflete o compromisso em desenvolver soluções completas e práticas, oferecendo uma plataforma interativa para compradores, vendedores e administradores no setor energético.

🚀 Starbound Astronaut
Starbound Astronaut é um jogo de plataforma 2D desenvolvido como parte da cadeira de Desenvolvimento de Jogos no Instituto Politécnico de Viseu. Neste jogo, controlas um astronauta que navega pelo espaço, superando obstáculos e coletando diamantes enquanto evitas cair das plataformas. Se caíres, perdes vida, mas reapareces. O objetivo final é alcançar o portal que te leva ao próximo nível, onde inimigos em movimento te aguardam.

🎮 Jogabilidade

  • Objetivo: Navegar através de plataformas espaciais, coletar diamantes, evitar quedas e alcançar o portal para avançar para o próximo nível.
  • Níveis: Dois níveis, cada um progressivamente mais desafiador, com inimigos introduzidos no segundo nível.
  • Inimigos: No segundo nível, os inimigos movem-se e tentam reduzir tua saúde quando estás por perto.
  • Reaparecer: Cair de uma plataforma faz com que o astronauta reapareça, mas perdes saúde.

🎮 Angry John
Angry John é um jogo interativo em 3D desenvolvido usando Unity e C#. Este projeto foi criado como parte da cadeira de Desenvolvimento de Jogos no Instituto Politécnico de Viseu. O jogo imerge os jogadores numa aventura emocionante, onde John, um valente macaco, combate robôs chamados "Devoradores de Vida" para restaurar o equilíbrio numa selva mística.

📖 Visão Geral do Projeto
Angry John é um jogo de ação-aventura de ritmo acelerado, para um jogador, onde os jogadores guiam John através de uma selva dominada por inimigos robóticos. O objetivo do jogo é derrotar esses robôs, resgatar os amigos de John e restaurar a vida na selva.

✨ Principais Características:

  • 🖥️ Compatibilidade de Plataforma: Disponível para Windows e macOS.
  • 🎮 Motor de Jogo: Desenvolvido em Unity utilizando C#.
  • 📜 Enredo: Uma selva mística invadida por robôs que drenam a energia vital.
  • 🎨 Design do Jogo: Combina elementos orientados pela história com desafios equilibrados.

🕹️ Mecânicas do Jogo

  • 🚶‍♂️ Movimento do Jogador: Controlo suave e responsivo para navegar pelo mundo em 3D.
  • ⚔️ Sistema de Combate: Engage em combate com lançadores de granadas e armas a laser.
  • 🧩 Desafios: Resolva quebra-cabeças e derrote inimigos para progredir.

🔍 Como Jogar

  • ⬇️ Download e Instalação: Baixe o jogo aqui para a sua plataforma.
  • 🎮 Jogar Online: Jogue a versão WebGL diretamente no seu navegador.
  • 🎮 Controlo:
    • WASD para movimento.
    • Espaço para saltar.
    • Clique Esquerdo para ataques.
    • Shift para esquivar.

🏆 Objetivo: Resgatar os amigos de John, derrotar robôs e restaurar a energia vital da selva.

💻 Instalação

  • ⚙️ Requisitos:
    • Windows: Versão 7 ou superior.
    • MacOS: Versão 10.12 ou superior.
    • Unity: Construído com Unity 2021.3.x.
  • 🚀 Executar o Jogo?:
    • 🌐 Versão WebGL: Jogue o jogo diretamente no navegador usando o link WebGL fornecido.
    • 📂 Versão Executável: Baixe o executável para a sua plataforma a partir do link fornecido e execute o jogo localmente.

🔨 Desenvolvimento

  • 📅 Marco do Projeto:
    • 📝 Ideia de Design de Jogo (IDJ): Conceituado a ideia de Angry John como um jogo de ação rápido.
    • 🎤 Apresentação de Pitch: Apresentada a história e as mecânicas do jogo.
    • 📋 Documento de Design de Jogo (DDJ): Detalhada a estrutura do jogo, objetivos e regras.
  • 🛠️ Ferramentas Utilizadas:
    • Unity: Para desenvolvimento de jogos e interação 3D.
    • C#: Linguagem de programação para lógica do jogo.
    • Blender: Modelagem e animação 3D.
    • Audacity: Criação e edição de áudio.

👥 Equipa: Este projeto foi desenvolvido em grupo na cadeira de Desenvolvimento de Jogos no Instituto Politécnico de Viseu.
